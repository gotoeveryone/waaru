# waaru

## Requirements

- Node.js
- Yarn
- Firebase CLI

## Setup

```console
$ yarn install
```

### Run

```console
$ yarn run serve
```

### Deploy

```console
$ yarn run build
$ firebase deploy
```

### Run your tests

```console
$ yarn run test
```

### Lints and fixes files

```console
$ yarn run lint
```

### Run your unit tests

```console
$ yarn run test:unit
```
