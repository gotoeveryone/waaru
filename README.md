# waaru

![Deploy Status](https://github.com/gotoeveryone/waaru/workflows/Deploy/badge.svg)

## Requirements

- Node.js 18.x
- Firebase CLI

## Setup

```console
$ npm install
```

### Run

```console
$ cp .env.example .env
$ npm run dev
```

### Deploy

```console
$ npm run build
$ firebase deploy
```

### Run your tests

```console
$ npm run test
```

### Run Lints

```console
$ npm run lint
```

### Run Formats

```console
$ npm run format
```
