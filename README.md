# waaru

## Requirements

- Node.js
- Firebase CLI

## Setup

```console
$ npm install
```

### Run

```console
$ npm run serve
```

### Deploy

```console
$ npm run build
$ firebase deploy
```

### Run your tests

```console
$ npm run test
```

### Lints and fixes files

```console
$ npm run lint
```

### Run your unit tests

```console
$ npm run test:unit
```
